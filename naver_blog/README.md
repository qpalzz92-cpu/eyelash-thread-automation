# 네이버 블로그 임시저장 자동화 (로컬 실행)

내 노트북에서 실행하면, 글 파일(`posts/*.txt`)의 **제목 + 본문을 네이버 블로그 글쓰기에
자동으로 채워 넣고 임시저장**까지 해주는 스크립트입니다. 사진은 임시저장된 글을 열어
직접 넣으면 됩니다.

## ⚠️ 왜 "내 노트북"에서만 돌려야 하나요?

네이버는 **낯선 기기·낯선 위치에서 로그인하면 보안 잠김**을 겁니다. 클라우드/서버(GitHub
Actions 등)에서 돌리면 바로 그 잠김이 걸려요. 그래서 이 스크립트는 **평소 쓰는 내
노트북에서, 내 인터넷으로** 실행하도록 만들었습니다.

- 로그인은 **최초 1회만 직접** 합니다. 세션(쿠키)은 `naver_blog/.profile` 에 저장돼
  다음부터는 자동 로그인돼요.
- **비밀번호는 코드/파일 어디에도 저장하지 않습니다.** (`.profile` 폴더는 git에 안 올라감)

## 준비물 (최초 1회 세팅)

1. **파이썬 설치** (3.10 이상) — https://www.python.org/downloads/
   설치할 때 "Add Python to PATH" 체크.
2. 터미널(명령 프롬프트)을 열고 이 폴더로 이동:
   ```
   cd 내려받은경로/eyelash-thread-automation/naver_blog
   ```
3. 필요한 것 설치:
   ```
   pip install -r requirements.txt
   playwright install chromium
   ```

## 실행

### 1) 입력만 하고 저장은 내가 직접 (권장 · 기본값)

```
python save_draft.py posts/속눈썹연장창업.txt --blog-id promote3404
```

- 크롬 창이 열립니다.
- (최초 1회) 로그인 화면이 뜨면 **직접 로그인**하세요. 끝나면 자동으로 이어집니다.
- 제목·본문이 자동으로 채워집니다.
- 이제 브라우저에서 **사진을 (사진) 자리에 넣고**, **소제목을 굵게/크게** 잡은 뒤,
  우측 상단 **[저장]** 을 눌러 임시저장하세요.
- 다 됐으면 터미널에서 **Enter** 를 눌러 브라우저를 닫습니다.

### 2) 임시저장까지 전부 자동

```
python save_draft.py posts/속눈썹연장창업.txt --blog-id promote3404 --auto-save
```

- 제목·본문 입력 후 **[저장](임시저장)까지 자동**으로 눌러줍니다.
- ⚠️ **발행(공개)은 절대 하지 않습니다.** 항상 임시저장까지만.
- 사진은 나중에 블로그 → 글쓰기 → 저장한 글에서 열어 직접 넣으세요.

## 새 글 추가하는 법

`posts/` 폴더에 아래 형식으로 `.txt` 파일을 만들면 됩니다.

```
[제목]
여기에 제목 한 줄

[본문]
여기부터 본문...
빈 줄은 그대로 문단 간격이 됩니다.
(사진) 이렇게 써두면 나중에 사진 넣을 자리 표시로 남습니다.
```

그리고 실행:
```
python save_draft.py posts/새글파일.txt --blog-id promote3404
```

## 자주 나는 문제

- **로그인 화면에서 안 넘어가요** → 로그인 완료 후 잠시 기다리면 자동 진행됩니다.
  캡차(보안문자)가 뜨면 직접 풀어주세요.
- **"제목/본문 입력 영역을 찾지 못했습니다"** → 네이버가 에디터 구조를 바꾼 경우입니다.
  `last_run.png`(실패 화면 스크린샷)와 함께 알려주시면 셀렉터를 업데이트해 드립니다.
- **저장 버튼을 자동으로 못 찾음** → 기본 모드로 실행해 직접 [저장]을 누르면 됩니다.

## 중요 주의사항

- 이 자동화는 **내 계정, 내 글, 임시저장**만 다룹니다. (개인 사용 목적)
- 저품질/누락 계정은 **글 발행 전 계정 예열**(목요일 전 정상 로그인 1회)을 먼저 하세요.
- `.profile/` 폴더에는 로그인 세션이 들어 있으니 **남에게 공유하지 마세요.**
