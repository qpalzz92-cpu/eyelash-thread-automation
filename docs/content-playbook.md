# 속눈썹 창업 스레드 콘텐츠 플레이북

> 예비 속눈썹 창업자(예비 원장)를 타깃으로 하는 스레드(Threads) 정보성 콘텐츠 제작 공식.
> 실제 조회수가 잘 나오는 스레드 콘텐츠의 알고리즘 신호 · 후킹 · 말투 · 멘트 구조 · 댓글 유도법을
> 리서치해 정리했으며, 앞으로 모든 콘텐츠 제안은 이 문서를 기준으로 한다.

---

## 0. 타깃 정의

- **핵심 타깃**: 속눈썹 창업을 "생각 중"인 사람 (아직 시작 안 함)
  - 직장인 → 기술직 전향 고민
  - 미용 자격증 준비 중 / 학원 수강 중
  - 이미 시술은 하지만 샵 오픈은 고민 중인 아이리스트
- **이들의 진짜 고민 (콘텐츠 소재의 원천)**
  1. "자격증 뭐부터 따야 해? 민간자격증만 있어도 되나?" (→ 국가면허 필요)
  2. "창업 비용 실제로 얼마 드나?" (교육비/인테리어/재료/월세)
  3. "손님이 안 오면 어쩌지?" (마케팅·단골 확보 불안)
  4. "지금 배워서 늦지 않았나? 포화 아닌가?"
  5. "혼자 1인샵 vs 직원 두기 vs 홈케어(공유샵)"
  6. "펌까지 배워야 하나? 연장만으로 되나?"

> 콘텐츠는 항상 **이 6가지 고민 중 하나를 정확히 찌르는 것**에서 출발한다.

## 1. 알고리즘이 실제로 보는 신호 (2026 기준)

리서치로 확인한, 스레드가 도달(조회수)을 결정하는 신호 우선순위.

| 우선순위 | 신호 | 우리가 할 일 |
|---|---|---|
| 1 | **첫 줄 체류시간(dwell time)** — 첫 줄 보고 몇 %가 스크롤을 멈추나 | 첫 150자 안에 후킹. 약하면 본문이 좋아도 추천 중단 |
| 2 | **비(非)팔로워 반응** — 추천탭·검색 유입이 반응하면 "바이럴 후보"로 재분류 | 뾰족한 주제 일관성으로 추천 노출 유도 |
| 3 | **첫 1시간 반응** — 초반 반응이 전체 도달을 결정 | 타깃이 깨어있는 시간에 발행(아래 참고) |
| 4 | **댓글 · 저장 · 재게시** — 좋아요보다 강한 신호 | 본문에 호기심, **댓글에 정답/부가정보** 배치 |
| 5 | **주제 일관성** — 계정이 한 주제로 뾰족한가 | 속눈썹·창업 외 잡다한 주제 섞지 않기 |

### 발행 시간대 (예비 창업자 타깃 기준)
- **1순위**: 밤 21:00~23:00 (퇴근 후 "나도 기술 배워볼까" 검색하는 시간)
- **2순위**: 점심 12:00~13:00, 출근 07:30~08:30
- 첫 1시간 반응이 중요하므로 **내가 댓글에 바로 응답할 수 있는 시간**에 올린다.

## 2. 말투 (Tone of Voice)

조회수 잘 나오는 스레드 계정의 공통 말투.

- **반말 아닌 "친한 언니 톤"의 해요체** — "~했어요", "~더라고요", "~하지 마세요"
- **1인칭 경험담** — "제가 3년 전에요", "저도 처음엔" → 정보를 경험으로 포장
- **단정 + 근거** — "이거 모르면 100% 망해요. 이유는요," 처럼 단정 후 바로 근거
- **전문용어는 풀어서** — "클래식/볼륨" 같은 용어는 초보 눈높이로 한 번씩 설명
- **광고 냄새 제거** — "우리 학원 오세요"(X) → "저는 이렇게 했어요"(O). 판매는 마지막에 살짝
- 문장은 짧게, **한 줄 = 한 호흡**. 줄바꿈 많이.

### 금지
- 딱딱한 정보 나열 ("속눈썹 창업 3단계: 1. 자격증 2. ...")
- 해시태그 도배
- "정보 공유합니다" 같은 밋밋한 오프닝

## 3. 후킹 공식 (첫 줄 = 전부)

첫 줄에 **숫자 · 의외성 · 질문 · 손실회피** 중 최소 하나. 이상적으로 둘.

| 유형 | 공식 | 예시 (속눈썹 창업) |
|---|---|---|
| 숫자 | "N년/N만원/N%" | "속눈썹 창업, 초기비용 이거 몰라서 800만원 날렸어요" |
| 의외성 | 통념을 뒤집기 | "속눈썹 자격증 3개 딴 거, 다 필요 없었어요" |
| 질문 | 타깃의 속마음 대신 말하기 | "속눈썹 배우고 싶은데 지금 너무 늦었을까요?" |
| 손실회피 | "~하지 마세요 / ~전에 보세요" | "속눈썹 학원 등록하기 전에 이거 꼭 보세요" |
| 폭로/솔직 | "아무도 말 안 해주는" | "속눈썹 원장들이 창업 전엔 절대 말 안 해주는 것" |

> **테스트**: 첫 줄만 보고 "어? 뭔데?" 소리가 안 나오면 버린다.

## 4. 멘트 구조 (본문 골격)

검증된 스레드 실전 글자수는 **50~150자 체감**(최대 500자). 아래 4단 구조를 기본형으로.

```
[1줄] 후킹        ← 스크롤 멈추게 (숫자/의외성/질문/손실회피)
[2~3줄] 공감/맥락  ← "저도 그랬어요" / 타깃의 상황 묘사
[본문] 정보 3~5개  ← 진짜 도움되는 알맹이. 짧은 리스트로
[마무리] 반전/한줄  ← 인사이트 한 방 + 자연스러운 CTA
[댓글] 정답/추가정보 ← 본문에서 "댓글에 정리해둘게요"로 유도
```

### 구조 예시 (뼈대)
```
속눈썹 학원 등록 전에 이건 꼭 알고 가세요.   ← 후킹(손실회피)

저 3년 전에 민간자격증만 믿고 시작했다가
개원 신고에서 막혔거든요.                    ← 공감/맥락

정리하면요,
· 시술 영업은 '국가 미용면허'가 있어야 합니다
· 민간자격증은 '실력 증명'이지 영업허가가 아니에요
· 학점은행제로 하면 시험 1번으로 끝낼 수 있어요

순서 잘못 밟으면 6개월 그냥 날아가요.        ← 반전/인사이트

면허 → 신고 → 오픈 순서 표로 정리해서
댓글에 남겨둘게요.                           ← 댓글 유도 CTA
```

## 5. 댓글 · 저장 · 팔로우 유도 (전환 설계)

리서치 결론: **본문에 정보를 다 주지 말고, 결정적 조각을 댓글로 내린다.** 체류시간↑ + 알고리즘 점수↑.

### 댓글 유도 (가장 강력한 순서대로)
1. **부가정보를 댓글에 배치** — "실제 비용 표는 댓글에 정리했어요" (최강)
2. **가벼운 선택 질문** — "1인샵 vs 공유샵, 어디로 시작하실 거예요?"
3. **경험 소환** — "속눈썹 배우다 포기할 뻔한 순간 있으면 댓글 주세요"

### 저장 유도
- "이건 캡처해두세요", "나중에 창업할 때 꺼내볼 리스트예요" 처럼 **저장 이유를 명시**

### 팔로우 전환
- 좋은 글만으론 팔로우 안 됨 → **직접 CTA 필요**
  - "속눈썹 창업 준비 과정 하나씩 올려요. 같이 준비하실 분 팔로우요."
- 프로필이 뾰족해야 함: "마케터"(X) → "예비 속눈썹 원장 옆에서 창업까지 같이 가는 계정"(O)
- 남의 글에 **2~3줄 진심 답글** 달기 → 그 답글 보고 프로필 유입

## 6. 콘텐츠 카테고리 (발행 로테이션)

한 주제만 반복하면 지루, 너무 흩어지면 알고리즘 불리. 아래 4카테고리를 로테이션.

| 카테고리 | 목적 | 예시 소재 |
|---|---|---|
| 정보/가이드 | 저장·신뢰 | 자격증 순서, 창업 비용, 인테리어, 재료 고르기 |
| 공감/현실 | 댓글·도달 | "창업 전 나 vs 후 나", 초보 원장 실수담 |
| 반전/직언 | 바이럴 | "속눈썹 포화라는 말, 반은 틀렸어요" |
| 전환/영업 | 문의·전환 | 무료 상담/체크리스트/전자책 등 (전체의 20% 이하) |

> 비율 권장: 정보 40% / 공감 30% / 반전 20% / 전환 10%.
> 영업 글이 20%를 넘으면 도달이 떨어진다.

## 7. 발행 전 체크리스트

- [ ] 첫 줄만 보고 멈칫하는가? (숫자/의외성/질문/손실회피 중 1+)
- [ ] "친한 언니 해요체" + 1인칭 경험담인가?
- [ ] 광고 냄새 없는가? (판매는 마지막 20%만)
- [ ] 본문에 다 주지 않고 **결정적 조각을 댓글로** 내렸는가?
- [ ] 저장/댓글/팔로우 중 **하나의 CTA**가 자연스럽게 있는가?
- [ ] 타깃이 깨어있는 시간(밤 9~11시)에 발행하고, 초반 댓글에 응답 가능한가?
- [ ] 속눈썹·창업 주제 일관성이 유지되는가?
