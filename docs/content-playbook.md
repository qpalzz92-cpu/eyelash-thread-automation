# 1인 뷰티샵 스레드 콘텐츠 플레이북

> **타깃: 뷰티 창업 준비생 + 1인샵 원장님** (속눈썹·네일·왁싱·반영구·피부관리 등)
> 조회수 잘 나오는 스레드 콘텐츠의 알고리즘 신호·후킹·말투·멘트 구조·댓글 유도법을
> 리서치해 공식화했다. 앞으로 모든 콘텐츠 제안은 이 문서를 기준으로 한다.

---

## 0. 타깃 정의

두 층을 함께 노린다. 하나의 글이 둘 다 찌를 수 있으면 최고.

### A. 뷰티 창업 준비생 (아직 시작 안 함)
- 직장인 → 기술직 전향 고민 / 자격증 준비 중 / 배웠는데 오픈은 망설이는 사람
- 업종: 속눈썹, 네일, 왁싱, 반영구, 피부관리, 헤어 등

### B. 1인샵 원장님 (이미 운영 중)
- 혼자 다 하느라 지치고, 손님·매출·마케팅이 늘 불안한 원장
- "기술은 되는데 손님이 안 늘어요"가 핵심 고민

### 이들의 진짜 고민 (콘텐츠 소재의 원천)
1. **창업 전** — 자격증/면허 순서, 창업 비용, "지금 늦었나·포화 아닌가", 1인샵 vs 공유샵
2. **손님** — 신규 유입, 재방문·단골화, 노쇼·리터치 관리
3. **매출/단가** — 가격 어떻게 정하나, 객단가 올리기, 시술 메뉴 구성
4. **마케팅** — 인스타/스레드/지도 홍보, 리뷰 관리, 홍보가 제일 막막
5. **운영/멘탈** — 혼자라 쉬지 못함, 번아웃, 세금·신고, 확장(직원/2호점)

> 콘텐츠는 항상 **이 고민 중 하나를 정확히 찌르는 것**에서 출발한다.
> 업종 특화(속눈썹 연장·펌 등)와 업종 공통(1인샵 운영·마케팅)을 번갈아 다룬다.

## 1. 알고리즘이 실제로 보는 신호 (2026 기준)

스레드가 도달(조회수)을 결정하는 신호 우선순위.

| 우선순위 | 신호 | 우리가 할 일 |
|---|---|---|
| 1 | **첫 줄 체류시간(dwell time)** | 첫 150자 안에 후킹. 약하면 본문 좋아도 추천 중단 |
| 2 | **비(非)팔로워 반응** — 추천·검색 유입이 반응하면 "바이럴 후보"로 재분류 | 뾰족한 주제 일관성으로 추천 노출 유도 |
| 3 | **첫 1시간 반응** | 타깃이 깨어있는 시간에 발행 |
| 4 | **댓글·저장·재게시** (좋아요보다 강함) | 본문에 호기심, **댓글에 정답/부가정보** |
| 5 | **주제 일관성** | 뷰티 창업·1인샵 운영 외 잡다한 주제 섞지 않기 |

### 발행 시간대 (뷰티 원장·예비 창업자 기준)
- **1순위**: 밤 21:00~23:00 (마감 후·퇴근 후 "나도 해볼까" 검색 시간)
- **2순위**: 점심 12:00~13:00, 출근 07:30~08:30
- 첫 1시간 반응이 중요 → **내가 댓글에 바로 응답할 수 있는 시간**에 발행.

## 1-1. 브랜드 페르소나 (화자 설정)

- **정체성**: "월 70만원 직원 → 뷰티샵·교육·유통·17개 가맹까지 만든 원장"
- **시그니처 주제**: **단가(객단가)를 높이는 뷰티샵 운영법** (싸게 경쟁 안 하고 비싸게 파는 법)
- **톤**: 자랑이 아니라 **"나도 바닥에서 왔다" 공감 + 지금은 알려주는 멘토**
- "70만원 → 자유" 전환 스토리는 최강 후킹 자산 → 콘텐츠에 반복해서 녹인다.
- **최종 퍼널**: 팔로우 → 신뢰 → **전자책 판매**. (판매 글은 전체의 10~20%만)
- 프로필/바이오 세팅은 `content/profile.md` 참고.

## 2. 말투 (Tone of Voice)

- **"친한 언니 톤"의 해요체** — "~했어요", "~더라고요", "~하지 마세요"
- **1인칭 경험담** — "제가 1인샵 3년 하면서", "저도 처음엔" → 정보를 경험으로 포장
- **단정 + 근거** — "이거 모르면 손님 끊겨요. 이유는요,"
- **전문용어는 풀어서** — 업종 용어(클래식/볼륨, 젤/아크릴 등)는 눈높이로 한 번씩 설명
- **광고 냄새 제거** — "우리 학원/제품 사세요"(X) → "저는 이렇게 했어요"(O). 판매는 마지막에 살짝
- 문장은 짧게, **한 줄 = 한 호흡**. 줄바꿈 많이.

### 금지
- 딱딱한 정보 나열 / 해시태그 도배 / "정보 공유합니다" 같은 밋밋한 오프닝

## 3. 후킹 공식 (첫 줄 = 전부)

첫 줄에 **숫자·의외성·질문·손실회피·폭로** 중 최소 하나, 이상적으로 둘.

| 유형 | 공식 | 예시 |
|---|---|---|
| 숫자 | "N년/N만원/N%" | "1인샵 초기비용, 이거 몰라서 800만원 더 썼어요" |
| 의외성 | 통념 뒤집기 | "자격증 3개 딴 거, 다 필요 없었어요" |
| 질문 | 타깃 속마음 대신 말하기 | "뷰티 배우고 싶은데 지금 너무 늦었을까요?" |
| 손실회피 | "~하지 마세요 / ~전에 보세요" | "1인샵 오픈하기 전에 이거 꼭 보세요" |
| 폭로 | "아무도 말 안 해주는" | "원장들이 창업 전엔 절대 말 안 해주는 것" |

> **테스트**: 첫 줄만 보고 "어? 뭔데?" 소리가 안 나오면 버린다.

---

## 4. 멘트 구조 (본문 골격)

검증된 스레드 실전 글자수는 **50~150자 체감**(최대 500자). 4단 구조를 기본형으로.

```
[1줄] 후킹        ← 스크롤 멈추게 (숫자/의외성/질문/손실회피)
[2~3줄] 공감/맥락  ← "저도 그랬어요" / 타깃 상황 묘사
[본문] 정보 3~5개  ← 진짜 도움되는 알맹이. 짧은 리스트
[마무리] 반전/한줄  ← 인사이트 한 방 + 자연스러운 CTA
[댓글] 정답/추가정보 ← "댓글에 정리해둘게요"로 유도
```

## 5. 댓글·저장·팔로우 유도 (전환 설계)

핵심: **본문에 다 주지 말고, 결정적 조각을 댓글로 내린다.** 체류시간↑ + 알고리즘 점수↑.

### 댓글 유도 (강한 순서)
1. **부가정보를 댓글에** — "실제 비용/가격 표는 댓글에 정리했어요" (최강)
2. **가벼운 선택 질문** — "1인샵 vs 공유샵, 어디로 시작하실 거예요?"
3. **경험 소환** — "손님 끊겨서 멘붕 온 적 있으면 댓글 주세요"

### 저장 유도
- "캡처해두세요", "창업할 때 꺼내볼 리스트예요" 처럼 **저장 이유 명시**

### 팔로우 전환
- 좋은 글만으론 팔로우 안 됨 → **직접 CTA**: "1인샵 운영 노하우 하나씩 올려요. 같이 버티실 원장님 팔로우요."
- 프로필 뾰족하게: "마케터"(X) → "1인 뷰티샵 원장님 창업·운영 메이트"(O)
- 남의 글에 **2~3줄 진심 답글** → 그 답글 보고 프로필 유입

## 6. 콘텐츠 카테고리 (발행 로테이션)

| 카테고리 | 목적 | 예시 소재 |
|---|---|---|
| 정보/가이드 | 저장·신뢰 | 자격증 순서, 창업비용, 가격 정하기, 재료·장비 |
| 공감/현실 | 댓글·도달 | "창업 전 나 vs 후 나", 1인샵 번아웃, 초보 실수담 |
| 반전/직언 | 바이럴 | "포화라는 말 반은 틀렸어요", "신규보다 단골" |
| 전환/영업 | 문의·전환 | 무료 체크리스트/상담/전자책 (전체의 20% 이하) |

> 비율 권장: 정보 40% / 공감 30% / 반전 20% / 전환 10%.
> **업종 밸런스**: 속눈썹 등 특정 업종 특화 글과, 업종 공통(1인샵 운영) 글을 번갈아.
