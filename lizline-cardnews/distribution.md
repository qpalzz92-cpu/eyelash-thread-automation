# 배포 문구 템플릿 (인스타 캡션 · 단톡방 · DM)

기존 배포 흐름 그대로:
**인스타 업로드(핵심 블러) → 단톡방에 링크 → 댓글/DM 요청 → 풀버전 DM**

- 인스타 업로드 시 **card_4는 `card_4_locked.png`(블러 버전)** 로 교체해서 올린다.
- 나머지 카드(1,2,3,5,6,7,8)는 그대로.
- 풀버전(블러 없는 card_4 포함)은 **공식 판매점/단톡 요청자에게만** DM.

## 1) 인스타 캡션 — 공통 뼈대
```
{첫 줄 훅}

원장님들이 실제로 쓰는 판매 노하우, 이번 편은 「{제목}」

· 저장해두고 상담 전에 한 번씩 보세요
· 4·5번 실제 멘트는 공식 판매점 전용 자료로 풀버전 제공해요

래쉬애딕트·아이원잇 공식 판매점 등록·문의 👉 프로필 링크 / DM (@리즈라인)

#속눈썹영양제 #래쉬애딕트 #아이원잇 #속눈썹연장 #속눈썹펌 #속눈썹샵 #뷰티창업 #1인샵 #리즈라인 #속눈썹연장창업
```

## 주제별 캡션 첫 줄(훅)
1. 업셀링 — `속눈썹 영양제 하나 팔고 끝? 객단가 2배 만드는 원장님은 여기서 '하나 더'를 붙입니다.`
2. 가격 응대 — `"비싸요"는 거절이 아니라 질문이에요. 여기서 무너지지 마세요.`
3. 재구매 — `영양제 매출의 진짜 승부는 '두 번째 구매'에서 갈립니다.`
4. 효과 없다 — `"효과 없어요" 하는 고객, 대부분 사용법 문제예요. 환불 말고 재구매로.`
5. 단독 수요 — `시술 안 하는 날에도 매출이 도는 샵, 뭐가 다를까요?`
6. 비대면 클로징 — `"생각해볼게요" 하고 나간 고객, 카톡 한 통으로 되살립니다.`
7. 후기 활용 — `잘 찍은 후기 1장이 광고 10개보다 셉니다.`
8. 오프닝 멘트 — `영양제는 설명이 아니라 '한 문장'으로 팔려요.`
9. 상태 응대 — `"상하셨네요" 이 한마디가 판매를 망칩니다.`
10. 매장 3접점 — `SNS 안 해도 영양제 잘 파는 샵의 비밀.`

## 2) 단톡방 공지 문구
```
[리즈라인 판매 노하우 #{번호}] {제목} 📌

이번 주 카드뉴스 올라갔어요!
· 인스타 👉 {게시물 링크}

4·5번 '실제 멘트/스크립트'는 블러 처리했어요.
전체 풀버전(멘트 원문) 필요하신 원장님은 이 글에 댓글 or DM 주세요 :)
바로 보내드릴게요 🙌
```

## 3) 풀버전 DM 문구
```
안녕하세요 원장님 😊
요청주신 「{제목}」 전체 자료(멘트 풀버전) 보내드려요!

· 카드 순서대로 저장해서 상담 때 참고하시면 좋아요
· 특히 4·5번 실전 멘트 + 7번 한 끗 포인트가 핵심이에요

적용해보시고 고객 반응 궁금하시면 언제든 공유해주세요 🙌
— 리즈라인 —
```

· 매주 판매 노하우 카드뉴스(멘트 풀버전) 자료

을 받아보실 수 있어요.

간단한 안내 도와드릴게요. 편하게 말씀 주세요!
