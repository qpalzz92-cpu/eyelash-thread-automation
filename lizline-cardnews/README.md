# 리즈라인 판매점 세일즈 카드뉴스 자동 생성

리즈라인 판매점 원장님 대상 **속눈썹 영양제(래쉬애딕트·아이원잇) 세일즈 카드뉴스**를
매주 자동 생성하는 파이프라인. 디자인(흰 배경·파랑/하늘색·Pretendard)은 고정,
**내용만 갈아끼우면** 카드가 자동으로 나온다.

## 구조
```
lizline-cardnews/
  generate.py       ← 렌더 엔진 (디자인 고정). 손댈 일 거의 없음
  content.py        ← 콘텐츠. 매주 여기만 수정한다
  distribution.md   ← 인스타 캡션 / 단톡방 / DM 배포 문구 템플릿
  fonts/            ← Pretendard woff2 (임베드용)
  out/<주제>/       ← 결과물
      card_1..8.png        풀버전 (DM·공식 판매점용)
      card_4_locked.png    인스타용 블러 버전 (실전 멘트 가림)
```

## 실행
```bash
cd lizline-cardnews
python3 generate.py        # 전체 주제 렌더
python3 generate.py 3      # 3번 주제만 렌더
```
- Chromium 헤드리스로 각 카드를 1080×1350 PNG로 렌더한다.
- 크롬 경로가 다르면 `CHROME_BIN` 환경변수로 지정.

## 매주 하는 일 (자동화 흐름)
1. `content.py`의 `TOPICS`에서 이번 주 주제 내용만 교체/추가.
2. `python3 generate.py` 실행 → 카드 이미지 자동 생성.
3. `out/<주제>/` 에서 인스타는 **card_4를 `card_4_locked`로 바꿔** 업로드,
   단톡/DM은 풀버전(card_1~8) 사용.
4. 캡션·단톡·DM 문구는 `distribution.md` 템플릿 사용.

## 카드 구성 (8장 고정 템플릿)
1 표지 · 2 문제/공감 · 3 핵심 포인트 · 4 실전 멘트(🔒블러) ·
5 체크포인트(자가진단) · 6 이번 주 적용해보기 · 7 한 끗 차이 · 8 CTA(공식 판매점 등록)

## 디자인 규칙
- 배경 흰색 / 본문 검정 / 제목·강조 **파랑(#2563EB)·하늘색(#38BDF8)**
- 제목 강조는 파란 하이라이트 박스, 항목은 둥근 카드 + 파란 체크
- CTA는 전 주제 공통: **공식 판매점 등록·문의 → 리즈라인**
