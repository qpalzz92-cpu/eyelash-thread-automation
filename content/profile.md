# 스레드 프로필 세팅 가이드

> 목적: 예비 원장·1인샵 원장이 프로필만 보고 "팔로우할 이유"를 3초 안에 느끼게 한다.
> 최종 퍼널: 팔로우 → 신뢰 → **전자책 판매**.

## 브랜드 페르소나 (모든 콘텐츠의 화자)

- **한 줄 정체성**: "속눈썹 10년 외길, 월 70만원 직원에서 뷰티샵·교육·유통·17개 가맹과 뷰티샵 자동화 시스템까지 만든 원장"
- **핵심 서사**: 기술만으로 번 게 아니다. 아무도 안 알려줘서 발로 뛰며 시행착오 → 자동화 시스템 구축. 그 과정을 자서전처럼 기록하며 막막한 원장님들과 나눈다.
- **시그니처 주제**: **단가(객단가)를 높이는 뷰티샵 운영법** — 싸게 경쟁하지 않고 비싸게 파는 법
- **톤**: 성공을 자랑하는 게 아니라, **"나도 바닥에서 왔다"는 공감 + 지금은 알려주는 멘토**
  - 자랑(X): "저 이만큼 벌어요"
  - 멘토(O): "월 70만원 받던 제가 이건 진짜 일찍 알았으면 했어요"
- 이 스토리(70만원 → 자유)는 **가장 강력한 후킹 자산**이니 콘텐츠에 반복해서 녹인다.

## 인스타랑 똑같이? → 이렇게 하세요

| 요소 | 인스타와 | 이유 |
|---|---|---|
| 핸들(아이디) | **통일** | "같은 브랜드"로 인지, 교차 유입 |
| 프로필 사진 | **통일** | 알아보기 쉽게 |
| 이름 | **통일** | 검색·기억 |
| 소개(바이오) | **스레드용으로 다르게** | 인스타 바이오는 예약·링크 위주, 스레드는 "팔로우 이유 한 줄"이 핵심 |

> 결론: 브랜드는 같게, **소개 문구만 스레드에 맞게 새로** 씁니다.

## 스레드 바이오 문구 (복붙용 3안)

인스타 원본:
> 🎉월 70만원 직원에서 > 경제적 자유
> 🌐샵, 교육, 유통 + 17개 가맹사업까지
> 🌳단가를 높이는 뷰티샵 운영 노하우

### A안 — 스토리 후킹형 (추천)
```
월 70만원 직원 → 뷰티샵·교육·유통·17개 가맹
🌳 단가 높이는 1인샵 운영법, 매일 하나씩
예비 원장·1인샵 원장 환영 🤍
```

### B안 — 가치 제안형
```
"싸게 말고 비싸게 파는" 뷰티샵 운영법
월 70만원에서 여기까지 온 진짜 이야기
창업·단가·단골 고민 → 댓글로 물어보세요
```

### C안 — 타깃 호명형
```
1인 뷰티샵 원장님 옆에서 같이 크는 계정
월 70 직원 출신이라 그 마음 다 알아요
단가 높이는 법 · 창업 순서 · 단골 만들기
```

> 세 개 다 스레드 규격(간결·2~3줄·팔로우 이유 명확)에 맞춤. A안부터 써보고 반응 보며 교체.

## 첫 고정(핀) 게시물

프로필 방문자가 "이 사람 뭐 하는 사람?"을 바로 알게, **자기소개 스토리 글**을 하나 써서 상단 고정.
- 구조: [후킹] 월 70만원 직원이었어요 → [전환점] 이때 뭘 깨달았나 → [지금] 뭘 하고, 여기선 뭘 나누는지 → [CTA] 팔로우 이유
- 초안은 요청 주시면 `content/posts.md`에 추가.

## 전자책 퍼널 (판매는 급하지 않게)

- 지금은 **팔로워·신뢰 쌓기 단계**. 전자책 직접 홍보는 전체 글의 10~20%만.
- 자연스러운 연결: 유용한 글 끝에 "이런 거 더 정리한 자료 있는데, 필요한 분?" → 댓글·DM 유도 → 리스트 확보
- 전자책은 프로필 링크 1개(인스타/링크트리)로만 걸어두고, 콘텐츠로 신뢰가 쌓이면 자연히 팔린다.
