# 원장 페르소나 핵심 자산 (사실 기록 · 콘텐츠 원천)

> 본인이 직접 준 사실. 모든 콘텐츠는 이걸 근거로 쓴다. (지어내지 않기)

## 지금까지 해낸 것 (자동화 4가지 = 실제 성과)
1. **안정화된 매뉴얼** → 내가 출근하지 않아도 샵과 교육이 돌아감
2. **가맹점 17개**로 확장
3. **제품 주문 자동답변 + 택배 접수** 자동화
4. **콘텐츠 자동 발행**

→ 한마디로 "내 몸(노동)이 아니라 시스템이 버는 구조".

## 앞으로 하려는 것 (미션 · 계정의 존재 이유)
- 쌓아온 노하우를 나눠서, **많은 1인샵 원장님들이 나처럼 덜 고생하고 빨리 자리잡게** 돕기.
- (최종 퍼널: 전자책 판매)

## 한 발 앞서 말해온 사람 (권위·시그니처 서사)
- "시술만 잘하면 돼" 할 때 → **마케팅**의 중요성을 말함
- 다들 "마케팅 잘해야 해" 할 때 → **브랜딩**의 중요성을 말함
- 다들 "브랜딩이 중요해" 할 때 → 결국 **시스템**이 가장 중요하다고 말함
- 순서: **마케팅 → 브랜딩 → 시스템**. 다 직접 겪어서 아는 것.

## 핵심 통찰 (반전 소재)
- 1인샵은 "안 돼서" 문 닫는다고들 생각하지만,
  **돈 잘 벌어도 문 닫는 샵이 많다.**
- 이유: **노동에만 의존**했기 때문. 내 몸이 멈추면 매출도 멈추는 구조라 오래 못 감.
- 결론: 끝까지 살아남는 건 **시스템(내가 없어도 돌아가는 구조)**을 만든 사람.

## 현재 위치 (중요 — 스토리 시점)
- **지금은 시술을 직접 하지 않음.** 현장(시술대)에서 내려온 지 오래.
- 노쇼·진상·객단가 고민 같은 **초기 단계는 이미 졸업**함.
- 스토리는 **회고·멘토 시점**으로: "그땐 나도 그랬지 → 지금은 시스템/매뉴얼/직원으로 다 돌아가 → 그때의 나 같은 원장님 보라고".
- 절대 "요즘 내가 시술하며 이 고생 중"처럼 쓰지 말 것. (현재진행 X, 과거회고 O)

## 콘텐츠 금지/주의 (가드레일)
- **직업을 몸 상하는 일로 묘사 금지** — "손목 시큰, 파스, 몸 망가진다" 같은 표현 X.
  (본인이 속눈썹 창업 교육을 하므로, 예비 수강생이 겁먹으면 부정적 영향.)
  → 힘듦은 '신체 손상'이 아니라 '혼자 다 떠안는 업무 과부하·외로움'으로 표현.
- 가맹은 대놓고 영업 X → "나랑 같이 가기로 한 원장도 있고" 식으로 사실처럼 씨앗만.

## ★ 콘텐츠 수준: '고수'만 (제일 중요)
- 화자는 10년·17개 가맹 전문가. **누구나 아는 뻔한 팁은 금지** (예: "예약금 걸어라", "리마인더 보내라", "후기 부탁해라"). 그런 건 오히려 하수처럼 보임.
- 반드시 **역발상 / 남들 모르는 한 끗 / 현장에서만 아는 인사이트**로.
  - 예: "예약금 세게 걸면 신규가 준다" / "취소를 쉽게 해줘야 노쇼가 준다" / "노쇼는 3종류, 전체 규정은 신규를 잃는다"
- 통념을 뒤집거나, 원인을 다르게 짚거나, 세그먼트로 쪼개는 식.

## ★ 본문 → 댓글 유도 (읽고 빠지지 않게)
- 본문을 '자기완결'로 끝내면 사람들이 읽고 나감. **핵심 팁/정답 하나는 본문에서 빼서 댓글에** 두고, 본문 끝에서 유도: "~는 댓글에 적어둘게 👇".
- 특히 데이터·팁 글에 필수. 스토리는 예외 가능.

## ★ 도메인 팩트 지어내기 금지 (제일 자주 하는 실수)
- 내가 '고수 인사이트'랍시고 지어내면 현장과 어긋남(예: 틀린 '텀' 이론). **확실치 않으면 일반화하거나 사용자에게 확인.** 사용자의 실제 경험이 항상 우선.
- 검증된 사용자 현장 팩트(노쇼 예시):
  - 장기(2주 뒤) 예약은 애초에 거의 안 함.
  - 노쇼·취소는 **전날·당일에 집중** (변심형이 주범).
  - **전날 취소 링크를 걸면 취소율 폭증** → 취소 문턱은 '적당히'(취소하려면 전화 한 통은 하게).
  - 예약금은 '환불 불가'보다 **'시술비에서 차감'**으로 안내하면 거부감↓.

## 길이 & 댓글 규칙
- **길이 다양화 필수.** 다 비슷하면 안 됨. 정보성/할 말 많은 주제는 길게(최대 500자 근처), 스토리는 중간, 데이터 반전은 짧고 강하게도.
- **댓글도 한 줄 금지.** 2~4줄로, 본문에 없는 '진짜 한 끗'을 더 얹기(디테일).

## 후보 A/B/C 규칙 (영진이 방식 = 형식을 다르게)
- 같은 주제로 **3가지 다른 형식**을 만든다 (후킹만 다른 게 아님):
  1. **스토리텔링** — 내 회고·경험, 감정 몰입 (스레드에서 제일 잘 터짐)
  2. **데이터/뉴스/논문 기반** — 통계·최신 자료 훅, 신뢰·저장
  3. **정보성(how-to)** — 실전 단계·리스트, 저장각
- 필요 시 정보성 대신 **반전/직언**을 섞어 도달 테스트.
- 각 세트에 ⭐추천 1개 표시. 성과 보고 다음 세트에 반영.

## 후보 브리핑(사용자→나) 최소 3요소
1. 주제  2. 내 진짜 경험·숫자 하나(핵심)  3. 느낌(세게/공감/데이터/반전)

## 정보성 글 공식 (뻔함 방지)
- 그냥 팁("가격에 이유를 붙여라")은 뻔해서 '한 발 앞선' 브랜드를 깎아먹음.
- **공식 = 진짜 데이터(훅) + 내 17샵 시야(근거) + 한 끗 다른 통찰**
  - 예: "미용업 3년 생존율 73% 1위(데이터) → 근데 4명 중 1명 폐업 → 17개 샵 보니 싸게 판 곳이 먼저 망하더라(내 시야) → 싼 가격은 원장을 지치게 한다(통찰)"
- 통계는 훅·근거로만 살짝. 논문/뉴스 나열은 딱딱해서 지양.
- 창업 교육 유치를 위해, 업종 매력(적은 자본·평생 기술·꾸준한 수요)을 자연스럽게 1~2줄 섞을 수 있음.

## 참고 통계 (검증됨, 2025)
- 미용실 3년 생존율 73.4% (전 업종 1위, 국세청 기반) / 1년 생존율 91.6%
- 자영업 전체 5년 생존율 40.2% (10곳 중 6곳 5년 내 폐업), 1년 내 폐업 22%

## 톤 유지
- 반말 회고·멘토 톤. 자랑 X, "나도 겪어봤다" 공감 + 한 발 앞선 통찰.

## ★★ 터지는 스레드 6원칙 (2026-07, 제일 최신 · 모든 글에 적용)
> 조회수는 첫 줄, 완독은 본문, 도달은 저장·공유, 대화는 댓글이 만든다.

### 1) 첫 줄(후킹) 먼저 기획 — 본문보다 훅부터
- 7가지 훅 유형: 숫자형 · 손해형 · 반전형 · 비밀공개형 · 정보성형 · 데이터형 · 실패경험형.
- 다 쓰지 말고 그 글에 제일 맞는 **베스트 3개만**. 합치거나 버려도 됨.
  - 예: 숫자형+손해형 합치기 / 데이터 없으면 데이터형 버리기 / 실패경험보다 손해형 우선.
- 첫 줄 하나로 클릭이 갈린다. 무조건 첫 줄부터 짠다.

### 2) 완독 흐름 — 끝까지 읽게
- 본문 구조: **실패경험 → 반전 → 이유 → 해결법 → 결론**.
- 첫 줄이 클릭을 만들고, 본문 흐름이 완독을 만든다. 중간에 빠질 틈 없게.

### 3) 저장되는 정보만 — 저장이 도달을 부른다
- 저장각 넣기: **체크리스트 / 복붙 가능한 문구·프롬프트 / 실수하기 쉬운 포인트**.
- "나중에 다시 봐야지" 싶은 걸 하나는 꼭.

### 4) 공유되는 문장 + 왜 공유되는지
- 공유 유발 문장을 **원장님 타겟**으로: 예) "이거 때문에 고생하는 원장님 있으면 공유해줘!"
- 이유: 원장님들은 '나 같은 원장에게 도움될 것'을 공유함 → 원장 타겟 문장이 공유를 부른다.

### 5) 댓글 달리는 마무리 — 경험 꺼내게
- 끝은 **경험을 꺼내게 하는 질문**. 매번 다르게.
  - "다들 이런 경험 있으면 하나만 얘기해줘" / "이 방법 어때? 더 좋은 방법 있으면 나도 알려줘" 등.

### 6) 조회수 100만 시리즈 — 하나로 안 끝냄
- 잘 나온 주제는 **같은 주제 후속 스레드(시리즈)**로 이어서 만든다.
- 시리즈 표기: title/topic에 (1/n 시리즈) 식으로. 후속은 앞 글과 각도만 다르게.

> 위 6원칙은 앞서 정한 고수/역발상·본문→댓글 유도·길이 다양화·팩트 금지와 **함께** 지킨다(대체 아님).

## 제목 표기 규칙 (노션 게시판)
- 제목 앞에 카테고리/형식 단어(마케팅·데이터·정보·스토리…) **쓰지 않음** — 이미 유형·카테고리 칸에 있음.
- 대신 **주제 그룹 번호**로 같은 주제 3형식을 한눈에 묶는다: `이모지 N. 본제목`.
  - 예: `📖 2. 예약금 세게 걸었더니…` / `📊 2. 노쇼는 '전날'에 몰린다` / `📝 2. 노쇼는 3종류다` (셋 다 주제2=노쇼)
  - 후속 시리즈(2편)도 같은 주제 번호를 쓰고 본제목에 'N편' 표기.
- 이모지는 형식 힌트로 유지: 📖 스토리 / 📊 데이터 / 📝 정보 / 📣 마케팅. ⭐는 추천 글에만.
